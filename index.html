Frequently Asked Questions

Everything you need to know about TARS

How does TARS understand what's on my screen?

TARS uses advanced computer vision and natural language processing to analyze webpage content. It can identify forms, buttons, text fields, and other interactive elements, understanding their context and purpose to take appropriate actions.

Is my data secure with TARS?

Absolutely. TARS processes everything locally in your browser. Your OpenAI API key is stored securely in your browser's local storage and never transmitted to our servers. We prioritize your privacy and security above all else.

Do I need an OpenAI API key?

Yes, TARS requires an OpenAI API key to function. This ensures you have full control over your AI usage and costs. You can get an API key from OpenAI's website and configure it in TARS settings.

Which websites does TARS work on?

TARS works on virtually any website. It's designed to understand and interact with standard web elements across all sites, from simple forms to complex web applications.

Can I control what actions TARS takes?

Yes! You can enable or disable auto-execution in the settings. When auto-execution is off, TARS will ask for confirmation before taking any action. You have complete control over what it does.

How much does TARS cost?

TARS is completely free to use! You only pay for your OpenAI API usage, which typically costs just a few cents per interaction. There are no subscription fees or hidden costs.
